# resume-template

a resume template that landed multiple big-tech and sf startup software engineering offers.
based on the most accomplished individual ever.
built for undergraduate cs students.

## files

- `Resume-Template.pdf` — formatted template (view/download)
- `resume-template.tex` — latex source
- `resume-image.png` — preview

## usage

1. open `resume-template.tex` in overleaf or your local latex editor
2. replace barney stinson's info with yours
3. export as pdf — save all three: pdf, png, tex. helps with visibility.

## what works

- one page. hard rule.
- open source, research, publications section matters for research-oriented roles (mts/swe research)
- go deep on 2-3 experiences over spreading thin across 4-5
- bullets: [action verb] + [what you built] + [metric]
- every project lists its tech stack
- fill the page — no whitespace at the bottom
- gpa only if ≥ 3.5 (we don't need to talk about mine)
- links: github + linkedin, personal site if you have one

## why latex?

- formatting is consistent across every machine and pdf viewer
- version control friendly — tweak spacing and fonts without fighting a word processor
- it's kind of fun tbh

## why barney stinson?

- my favorite character of all time
- incredibly accomplished across a diverse portfolio of work
- his bullets are accidentally great resume writing — specific numbers, clear outcomes, strong verbs
